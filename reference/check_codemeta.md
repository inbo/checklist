# Check the package metadata

Use the checks from
[`codemetar::give_opinions()`](https://docs.ropensci.org/codemetar/reference/give_opinions.html).

## Usage

``` r
check_codemeta(x = ".")
```

## Arguments

- x:

  Either a `checklist` object or a path to the source code. Defaults to
  `.`.

## Value

A `checklist` object.
