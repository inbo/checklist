# Make your DESCRIPTION tidy

A tidy `DESCRIPTION` uses a strict formatting and order of key-value
pairs. This function reads the current `DESCRIPTION` and overwrites it
with a tidy version.

## Usage

``` r
tidy_desc(x = ".")
```

## Arguments

- x:

  Either a `checklist` object or a path to the source code. Defaults to
  `.`.
