# Ask for rights holder or funder

Ask for rights holder or funder

## Usage

``` r
ask_rightsholder_funder(org, type = c("rightsholder", "funder"))
```

## Arguments

- org:

  Organisation object

- type:

  Character, either `"rightsholder"` or `"funder"`

## Value

A list with the selected names and the updated organisation object
