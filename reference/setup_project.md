# Set-up `checklist` on an existing R project

Use this function to set-up or change the `checklist` infrastructure for
an existing project. The function interactively asks questions to set-up
the required checks.

## Usage

``` r
setup_project(path = ".")
```

## Arguments

- path:

  the project root folder
