# Write a `CITATION.cff` file

This file format contains the citation information. It is supported by
GitHub, Zenodo and Zotero. This function is super-seeded by
[`update_citation()`](https://inbo.github.io/checklist/reference/update_citation.md).

## Usage

``` r
write_citation_cff(x = ".", roles)
```

## Arguments

- x:

  Either a `checklist` object or a path to the source code. Defaults to
  `.`.

- roles:

  No longer used.

## Value

An invisible `checklist` object.
