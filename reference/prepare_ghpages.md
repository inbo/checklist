# Prepare a `gh-pages` branch with a place holder page

Prepare a `gh-pages` branch with a place holder page

## Usage

``` r
prepare_ghpages(x = ".", verbose = TRUE)
```

## Arguments

- x:

  Either a `checklist` object or a path to the source code. Defaults to
  `.`.

- verbose:

  display some progress info while downloading
